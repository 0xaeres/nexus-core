# The AI Engineer Book

A static, Rust Book-style technical site for AI engineering concepts.

## Open locally

Open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Host for free

This is a no-build static site. Upload the `ai-engineer-book` folder to any static host:

- GitHub Pages: push the folder contents to a repo and enable Pages from the main branch.
- Netlify: drag the folder into Netlify Drop.
- Cloudflare Pages: create a Pages project and select this folder as the output.

## Contents

The site covers Python services, LLM internals, prompting, RAG, tools, MCP, agents, memory, orchestration, guardrails, LLMOps, cloud deployment, and capstone architectures.
